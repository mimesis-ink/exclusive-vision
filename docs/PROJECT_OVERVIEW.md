# 独家视界：谎言必须死
## Exclusive Vision: Lies Must Die

### 项目概述 / Project Overview

这是一部短剧剧本类小说项目，融合了公关行业、鉴谎系统和商业阴谋等元素。

This is a short drama script novel project that combines elements of the PR industry, lie detection systems, and business conspiracies.

---

## 📊 篇幅规划 / Length Planning

- **总字数 / Total Words**: 200,000 - 250,000 字
- **总章节数 / Total Chapters**: 100 - 120 章（完结）
- **单章字数 / Words per Chapter**: 1,800 - 2,000 字（严格控制）
- **章节要求 / Chapter Requirements**: 
  - 开头有冲突 / Conflict at the beginning
  - 中间有反转 / Reversal in the middle
  - 结尾有钩子 / Hook at the end

---

## 🎬 核心设定 / Core Concept

### 主线故事 / Main Story
- **主角设定**: 公关女王 + 鉴谎系统（视网膜弹窗）
- **对抗势力**: 伪君子丈夫 + 资本阴谋

### 系统特色 / System Features
- **鉴谎系统**: 视网膜弹窗显示
- **UI描写要求**: 每章至少3-5次系统弹窗
- **特效亮点**: 这是核心卖点，也是未来短剧中最炫酷的特效部分

---

## 📚 整体结构 / Overall Structure

### 三卷式结构 / Three-Volume Structure

**第一卷：觉醒** (1-40章)
- 系统觉醒
- 发现真相
- 初步对抗

**第二卷：博弈** (41-80章)
- 深入调查
- 商业战争
- 关系裂痕

**第三卷：终局** (81-120章)
- 真相大白
- 最终对决
- 故事完结

---

## ✍️ 写作原则 / Writing Principles

### 禁止事项 / Prohibited
❌ 大段环境描写
❌ 抒情式心理描写
❌ 冗长的过渡段落

### 推荐写法 / Recommended
✅ 直接写动作
✅ 对话推动情节
✅ 系统UI弹窗描写
✅ 短句、快节奏

### 示例对比 / Examples

**❌ 避免这样写：**
> 窗外的雨淅沥沥地下着，像林浅此刻的心情一样沉重而复杂。她站在落地窗前，思绪万千...

**✅ 应该这样写：**
> 窗外下着暴雨。林浅点燃了一支烟，看着红色的烟头明灭。
