# 《独家视界：谎言必须死》
# 故事大纲 / Story Outline

## 核心设定

### 主角：林浅
- **身份**: 顶级公关公司CEO
- **年龄**: 30岁
- **特点**: 精明干练、理性冷静
- **能力**: 鉴谎系统（视网膜弹窗）
- **目标**: 揭穿真相、守护正义

### 反派：陆景深
- **身份**: 林浅的丈夫、投资公司总裁
- **表面**: 温柔体贴、事业有成
- **真相**: 伪君子、资本操控者
- **目标**: 利用林浅为其洗白

### 系统设定
- **名称**: 真相之眼（Truth Eye）
- **功能**: 
  - 谎言检测（准确率92%+）
  - 情绪分析
  - 关系网络可视化
  - 任务系统
  - 危机预警
- **显示**: 视网膜弹窗（只有林浅能看到）
