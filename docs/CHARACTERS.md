# 人物设定
## Character Profiles

## 主要角色

### 林浅（主角）

**基本信息**
- 姓名：林浅
- 性别：女
- 年龄：30岁
- 职业：顶级公关公司CEO（浅影公关）

**外貌特征**
- 身高：168cm
- 特点：职业干练、气质冷静
- 着装：简约高级的职业装

**性格特点**
- 核心：理性、冷静、观察力强
- 优点：专业能力强、决断力强、正义感
- 缺点：工作狂、不善表达情感、过度理性

**能力设定**
- 专业能力：顶级公关策划、危机处理
- 系统能力：鉴谎系统（视网膜弹窗）
- 核心技能：
  - 谎言检测（视网膜显示）
  - 情绪分析
  - 关系网络可视化
  - 危机预判

**人物弧光**
- 起点：成功的公关精英，幸福的婚姻（表面）
- 转折：获得系统，发现丈夫的谎言
- 成长：从职业女性到正义守护者
- 终点：揭穿真相，重塑人生

**经典台词**
- "谎言在真相面前，毫无意义。"
- "我不需要信任，我只相信证据。"
- "公关的本质是说服，但我更擅长揭露。"

---

### 陆景深（主要反派）

**基本信息**
- 姓名：陆景深
- 性别：男
- 年龄：33岁
- 职业：投资公司总裁（深景资本）
- 与主角关系：丈夫

**外貌特征**
- 身高：182cm
- 特点：成熟稳重、气质儒雅
- 着装：高级定制西装

**性格特点（双面）**

*表面人设：*
- 温柔体贴的丈夫
- 事业有成的精英
- 稳重可靠的男人

*真实面目：*
- 冷酷无情的资本操控者
- 精于算计的伪君子
- 为达目的不择手段

**核心动机**
- 利用林浅的公关能力为其洗白
- 通过婚姻获得社会地位和信任
- 掩盖商业犯罪行为

**谎言体系**
- 从婚姻开始就是精心设计的骗局
- 所有的温柔都是伪装
- 公司业务涉及金融诈骗、洗钱

**人物弧光**
- 起点：完美丈夫的伪装
- 中期：面具逐渐脱落
- 末期：真面目完全暴露
- 终点：受到法律制裁

**经典台词**
- "林浅，你永远不会知道真相。"
- "我所做的一切，都是为了我们的未来。"（谎言）
- "你以为你赢了？你只是我计划的一部分。"

## 人物成长线

### 林浅的成长
1. 职业精英 → 发现真相 → 正义战士 → 守护者
2. 信任丈夫 → 怀疑 → 调查 → 对抗 → 胜利
3. 依赖感情 → 理性觉醒 → 独立坚强

### 陆景深的堕落
1. 完美伪装 → 露出破绽 → 真相暴露 → 疯狂反击 → 失败

### 配角的转变
- 宋雨晴：助理 → 战友
- 江沉：记者 → 盟友
- 王秘书：帮凶 → 证人

---

*人物是故事的灵魂，每个角色都应该有自己的动机和弧光。*
