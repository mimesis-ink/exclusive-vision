# 章节结构模板 / Chapter Structure Template

## 标准章节组成

### 章节信息
```
章节编号: 第X章
章节标题: [标题]（简短有力，5-10字）
字数: 1,800-2,000字
```

---

## 📐 结构模板

### 第一部分：开头冲突（200-300字）

**目标**: 抓住读者注意力，建立本章核心冲突

**写作要点**:
1. 从动作或对话切入
2. 立即引入矛盾/冲突
3. 出现1次系统弹窗

**示例框架**:
```
[场景设定 - 10-20字]
[突发事件/对话 - 50-80字]
[系统弹窗 - 30-50字]
[主角反应 - 50-80字]
[冲突升级 - 50-80字]
```

---

### 第二部分：中间反转（1,200-1,400字）

**目标**: 展开情节，制造反转，推进故事

**写作要点**:
1. 情节逐步展开
2. 出现2-3次系统弹窗
3. 至少1个意外反转
4. 推进主线或揭示副线

**常见反转类型**:
- 发现新线索
- 信任的人说谎
- 计划被破坏
- 隐藏身份暴露
- 真相与表象相反

**示例框架**:
```
场景1（300-400字）:
[情节推进]
[对话/动作]
[系统弹窗1]
[新信息/线索]

场景2（400-500字）:
[冲突深化]
[对话/动作]
[系统弹窗2]
[小反转]

场景3（400-500字）:
[情节高潮]
[对话/动作]
[系统弹窗3]
[大反转/揭秘]
```

---

### 第三部分：结尾钩子（300-400字）

**目标**: 留下悬念，吸引读者继续阅读

**写作要点**:
1. 解决本章表面冲突
2. 引入新的问题/危机
3. 出现1次系统弹窗
4. 强烈的悬念感

**钩子类型**:
- 新危机出现
- 意外访客
- 重要发现
- 关键对话
- 系统警报

**示例框架**:
```
[本章冲突暂时解决 - 80-100字]
[过渡/喘息 - 50-80字]
[突发事件/新发现 - 80-100字]
[系统弹窗 - 30-50字]
[悬念结尾 - 50-80字]
```

---

## 💻 系统弹窗分配

### 每章3-5次，建议分配：

1. **开头部分**: 1次
   - 检测谎言/情绪
   - 引入冲突

2. **中间部分**: 2-3次
   - 揭示信息
   - 分析情况
   - 任务提示

3. **结尾部分**: 1次
   - 警告危险
   - 新线索
   - 任务更新

---

## 🎯 情节节点设计

### 每章必须包含：

1. **一个核心冲突**
   - 人物之间的矛盾
   - 主角的困境
   - 外部威胁

2. **一个信息增量**
   - 新线索
   - 新角色
   - 新关系
   - 真相碎片

3. **一个情绪波动**
   - 愤怒
   - 震惊
   - 恐惧
   - 怀疑
   - 释然

4. **一个行动决策**
   - 主角做出选择
   - 改变策略
   - 采取行动

## 📋 实战检查表

写作前：
- [ ] 明确本章要推进的剧情
- [ ] 设计好核心冲突是什么
- [ ] 想好反转点在哪里
- [ ] 规划好系统弹窗位置

写作中：
- [ ] 控制字数，不超不少
- [ ] 每100字检查一次节奏
- [ ] 确保对话简洁有力
- [ ] 系统弹窗准时出现

写作后：
- [ ] 统计字数（1,800-2,000）
- [ ] 检查三段式结构完整
- [ ] 确认系统弹窗3-5次
- [ ] 验证开头、反转、钩子
- [ ] 检查是否有注水内容
- [ ] 确认可拍性

```
字数进度     内容功能        节奏
0-200       开头冲突        快
200-500     情节展开        中
500-900     冲突深化        中快
900-1300    反转高潮        快
1300-1600   情绪沉淀        中
1600-1800   新悬念          快
1800-2000   钩子结尾        急
```

## 💡 实用技巧

### 1. 开头五种起式
- **对话起**: 从关键对话开始
- **动作起**: 从紧张动作开始
- **弹窗起**: 从系统警报开始
- **冲突起**: 从矛盾爆发开始
- **悬念起**: 从神秘事件开始

### 2. 中间三种转法
- **信息转**: 新信息颠覆认知
- **人物转**: 角色立场转变
- **局势转**: 优劣形势逆转

### 3. 结尾四种钩式
- **危机钩**: 新危险出现
- **秘密钩**: 重要秘密浮现
- **人物钩**: 关键人物登场
- **系统钩**: 系统重要提示
