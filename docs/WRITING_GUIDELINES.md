# 写作指南 / Writing Guidelines

## 📝 章节写作规范

### 一、字数控制

- **严格要求**: 每章 1,800 - 2,000 字
- **不可超出**: 避免单章过长影响节奏
- **不可过短**: 确保情节完整性

---

## 🎭 章节三段式结构

### 1. 开头（200-300字）- 冲突启动
- **核心任务**: 制造冲突、吸引读者
- **写作技巧**:
  - 从动作或对话开始
  - 立即引入矛盾点
  - 避免长篇铺垫

**示例开头**:
```
林浅刚走进办公室，手机就响了。
"林总，出事了。"助理的声音在发抖。
【系统提示：检测到情绪波动 - 恐惧指数87%】
```

### 2. 中间（1200-1400字）- 反转推进
- **核心任务**: 情节反转、制造悬念
- **写作技巧**:
  - 至少一个意外转折
  - 利用系统弹窗揭示真相
  - 推动主线或副线发展

**反转设计**:
- 看似简单的事件隐藏深层阴谋
- 信任的人说谎
- 计划被打乱需要重新应对

### 3. 结尾（300-400字）- 钩子悬念
- **核心任务**: 留下悬念、引导下一章
- **写作技巧**:
  - 提出新问题
  - 引入新角色/新信息
  - 制造危机感

**示例结尾**:
```
林浅关上电脑，准备下班。
门被推开，陆景深站在门口，脸色阴沉。
"我们需要谈谈。"
【系统提示：谎言检测 - 对象：陆景深 - 隐藏信息：重大】
```

---

## 💻 系统UI弹窗规范

### 每章必须出现 3-5 次

系统弹窗是本作品的**核心卖点**，也是未来短剧最炫酷的特效部分。

### 弹窗类型

#### 1. 谎言检测
```
【系统提示：谎言检测】
目标：陆景深
谎言指数：92%
关键词：项目、合作、资金
隐藏真相：已部分解锁
```

#### 2. 情绪分析
```
【系统提示：情绪分析】
目标：王秘书
真实情绪：焦虑（78%）、恐惧（65%）
表面情绪：镇定
评估：言行不一致，建议深度追问
```

#### 3. 关系网络
```
【系统提示：关系网络更新】
新增节点：张明 ←→ 陆景深（资金往来）
关联度：★★★★☆
风险等级：高
建议：立即调查
```

#### 4. 任务提示
```
【系统提示：新任务】
任务名：揭穿虚假项目
难度：★★★★★
时间限制：72小时
奖励：真相碎片×3
```

#### 5. 危机警报
```
【系统警报：危险！】
检测到恶意行为
来源：未知
威胁等级：紧急
建议：立即撤离
```

## 🚫 写作禁忌

### 1. 禁止注水式描写
**❌ 错误示例**:
> 天空中飘着几朵白云，阳光透过云层洒在大地上，给这座繁华的都市披上了一层金色的外衣。林浅走在街道上，看着来来往往的行人，心中思绪万千...

**✅ 正确示例**:
> 正午的阳光刺眼。林浅走出公司大楼，直奔对面的咖啡厅。

### 2. 禁止长篇心理活动
**❌ 错误示例**:
> 林浅心想，陆景深真的在骗我吗？我们结婚五年了，他对我那么好，怎么可能...不对，系统不会错，那他为什么要骗我？是因为那个项目吗？还是...

**✅ 正确示例**:
> 【系统提示：谎言检测 - 陆景深 - 92%】
> 林浅盯着视网膜上跳动的数字。五年婚姻，原来是场骗局。

### 3. 禁止冗长对话
**❌ 错误示例**:
> "林浅，你好啊，最近工作怎么样？我听说你们公司接了一个大项目..."
> "是啊，项目是挺大的，不过也挺累的，你知道的..."

**✅ 正确示例**:
> "你调查我？"陆景深声音冰冷。
> "你在撒谎。"林浅直视他的眼睛。
> 【系统提示：谎言检测 - 94% - 关键词：调查】

## ✨ 优秀写作模式

### 动作驱动
- 每个场景都要有具体动作
- 用动作展现人物性格
- 动作推动情节发展

### 对话推进
- 对话要有信息量
- 每句话都推动剧情
- 结合系统弹窗增加张力

### 系统互动
- 系统是主角的外挂也是助手
- 通过系统发现真相
- 系统提示引导情节

## 📋 章节检查清单

写完每一章后，请检查：

- [ ] 字数是否在 1,800-2,000 之间？
- [ ] 开头是否有明确冲突？
- [ ] 中间是否有至少一个反转？
- [ ] 结尾是否留下悬念钩子？
- [ ] 系统弹窗是否出现 3-5 次？
- [ ] 是否避免了大段环境描写？
- [ ] 是否避免了冗长的心理活动？
- [ ] 对话是否简洁有力？
- [ ] 情节是否适合短剧拍摄？
- [ ] 是否推进了主线或副线剧情？

---

## 🎬 短剧改编适配原则

### 可拍性
- 所有场景都要具体可拍
- 避免抽象的内心戏
- 系统弹窗作为特效呈现

### 节奏控制
- 快节奏、强冲突
- 3-5分钟一个高潮
- 每集留钩子

### 成本考虑
- 场景不要过于复杂
- 避免大场面描写
- 聚焦人物关系和情节

---

*记住：我们不是在写传统小说，而是在写可以直接拍摄的短剧剧本式小说！*
