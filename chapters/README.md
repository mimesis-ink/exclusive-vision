# 章节目录
# Chapters Directory

这个目录用于存放正式的小说章节。

## 📋 章节命名规范

章节文件应按以下格式命名：

```
chapter_001.md  - 第1章
chapter_002.md  - 第2章
chapter_010.md  - 第10章
chapter_100.md  - 第100章
```

## 📝 创建新章节

1. 复制模板：
```bash
cp ../templates/chapter_template.md chapter_XXX.md
```

2. 填写章节信息
3. 按照三段式结构创作
4. 确保包含3-5次系统弹窗
5. 检查字数（1800-2000字）

## 🗂️ 章节分卷

### 第一卷：觉醒（1-40章）
- chapter_001.md ~ chapter_040.md
- 主题：系统觉醒、发现真相、初步对抗

### 第二卷：博弈（41-80章）
- chapter_041.md ~ chapter_080.md
- 主题：深入调查、商业战争、关系裂痕

### 第三卷：终局（81-120章）
- chapter_081.md ~ chapter_120.md
- 主题：真相大白、最终对决、故事完结

## ✅ 章节检查清单

每章完成后请检查：

- [ ] 文件名格式正确（chapter_XXX.md）
- [ ] 字数在1800-2000之间
- [ ] 开头有明确冲突
- [ ] 中间有至少一个反转
- [ ] 结尾有悬念钩子
- [ ] 系统弹窗出现3-5次
- [ ] 无注水内容
- [ ] 适合短剧拍摄
- [ ] 推进了主线或副线剧情
