# 章节创作检查清单
## Chapter Writing Checklist

## 📋 使用说明

在完成每一章后，使用此清单进行自我检查。确保每一项都符合要求。

---

## ✅ 基础要求检查

### 字数控制
- [ ] 总字数在 1,800 - 2,000 字之间
- [ ] 开头部分：200 - 300 字
- [ ] 中间部分：1,200 - 1,400 字
- [ ] 结尾部分：300 - 400 字

### 结构完整性
- [ ] 有明确的"三段式"结构
- [ ] 开头、中间、结尾界限清晰
- [ ] 每个部分功能明确

---

## 🎭 情节检查

### 开头部分（冲突）
- [ ] 第一句话就有吸引力
- [ ] 在前50字内引入冲突
- [ ] 没有冗长的环境描写
- [ ] 没有长篇铺垫
- [ ] 直接进入故事

### 中间部分（反转）
- [ ] 至少有1个明显的反转
- [ ] 反转合理且出人意料
- [ ] 情节推进流畅
- [ ] 有新信息揭露
- [ ] 有人物互动或对抗

### 结尾部分（钩子）
- [ ] 留下明确的悬念
- [ ] 引发读者好奇
- [ ] 为下一章埋下伏笔
- [ ] 不是"生硬截断"
- [ ] 有情绪高点

---

## 💻 系统弹窗检查

### 数量要求
- [ ] 总共出现 3-5 次系统弹窗
- [ ] 开头部分至少 1 次
- [ ] 中间部分 2-3 次
- [ ] 结尾部分至少 1 次

### 质量要求
- [ ] 每次弹窗都有实际作用
- [ ] 弹窗揭示关键信息
- [ ] 弹窗推动情节发展
- [ ] 弹窗增加阅读张力
- [ ] 弹窗类型多样化

### 弹窗类型分布
- [ ] 使用了至少2种不同类型
- [ ] 谎言检测：___ 次
- [ ] 情绪分析：___ 次
- [ ] 关系网络：___ 次
- [ ] 任务系统：___ 次
- [ ] 危机警报：___ 次
- [ ] 其他：___ 次

---

## 🚫 内容质量检查

### 禁止内容（必须避免）
- [ ] ✓ 没有大段环境描写
- [ ] ✓ 没有抒情式心理活动
- [ ] ✓ 没有冗长的过渡段落
- [ ] ✓ 没有流水账式叙述
- [ ] ✓ 没有无意义的对话

### 示例检查

**环境描写**：
- [ ] ✓ 没有类似"窗外的雨淅沥沥地下着..."的描写
- [ ] ✓ 环境描写控制在一句话以内

**心理活动**：
- [ ] ✓ 没有长段的"林浅心想..."
- [ ] ✓ 心理活动通过对话或动作表现

**对话质量**：
- [ ] ✓ 对话简洁有力
- [ ] ✓ 每句对话都有信息量
- [ ] ✓ 对话推动情节发展

---

## 🎬 短剧改编检查

### 可拍性
- [ ] 所有场景都具体可拍
- [ ] 没有抽象的内心戏
- [ ] 动作描写清晰明确
- [ ] 对话适合演员表演

### 场景设置
- [ ] 场景不过于复杂
- [ ] 场景转换自然
- [ ] 避免大场面描写
- [ ] 聚焦人物和对话

### 节奏控制
- [ ] 快节奏，无拖沓
- [ ] 3-5分钟内能拍完
- [ ] 有明确的视觉呈现

---

## 📖 叙事检查

### 视角一致性
- [ ] 保持第三人称视角
- [ ] 主要跟随林浅的视角
- [ ] 视角切换合理

### 时间线
- [ ] 时间线清晰
- [ ] 与前一章衔接
- [ ] 时间跨度合理

### 信息量
- [ ] 揭示了新信息
- [ ] 推进了主线或副线
- [ ] 没有原地踏步
- [ ] 信息量适中（不过载）

---

## 👥 人物检查

### 人物塑造
- [ ] 主角行动符合性格
- [ ] 配角行为有动机
- [ ] 人物对话有个性
- [ ] 人物关系有发展

### 人物一致性
- [ ] 性格前后一致
- [ ] 行为逻辑合理
- [ ] 没有OOC（Out of Character）

---

## 🎯 与大纲对照

### 剧情推进
- [ ] 符合大纲规划
- [ ] 推进了本卷主题
- [ ] 为后续情节铺垫
- [ ] 没有偏离主线

### 本章定位
- [ ] 明确本章在整体中的位置
- [ ] 完成了本章应有的功能
- [ ] 与前后章节衔接

---

## ✍️ 语言文字检查

### 文字质量
- [ ] 没有错别字
- [ ] 没有语病
- [ ] 标点符号正确
- [ ] 句子通顺流畅

### 语言风格
- [ ] 简洁有力
- [ ] 符合短剧风格
- [ ] 没有过于文学化
- [ ] 适合快速阅读

### 节奏感
- [ ] 长短句结合
- [ ] 快慢节奏变化
- [ ] 有情绪起伏

---

## 🔍 细节检查

### 前后呼应
- [ ] 与前章情节衔接
- [ ] 伏笔和细节一致
- [ ] 人物状态连贯

### 逻辑自洽
- [ ] 情节逻辑合理
- [ ] 人物行为可信
- [ ] 时间空间一致

### 细节真实
- [ ] 专业描写准确（公关、商业等）
- [ ] 生活细节真实
- [ ] 避免常识性错误
