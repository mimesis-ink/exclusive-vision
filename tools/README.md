# 番茄小说网流量分析工具
# Fanqie Novel Website Traffic Analysis Tool

## 简介 / Introduction

这是一个用于分析《独家视界：谎言必须死》在番茄小说网流量情况的工具。

This tool analyzes the traffic status of "Exclusive Vision: Lies Must Die" on Fanqie Novel website (番茄小说网).

## 功能 / Features

- ✅ 检查小说在番茄小说网的发布状态
- ✅ 分析小说的平台适配度
- ✅ 评估流量潜力
- ✅ 生成详细的分析报告
- ✅ 提供发布建议

## 使用方法 / Usage

### 基本使用

```bash
# 进入工具目录
cd tools

# 运行分析工具
python3 analyze_traffic.py
```

### 输出示例

工具会生成包含以下信息的报告：

1. **发布状态**：小说是否已在番茄小说网发布
2. **流量情况**：是否有流量数据
3. **小说信息**：类型、标签、章节数、字数等
4. **平台适配度**：评分及适配原因
5. **流量潜力**：当前状态及发布建议
6. **总结建议**：综合分析结果

## 分析维度 / Analysis Dimensions

### 1. 平台适配度评估

工具会根据以下因素评估小说对番茄小说网的适配度：

- **类型匹配**：都市职场类型在番茄平台受欢迎
- **标签匹配**：公关、商业、系统、短剧等标签
- **短剧改编潜力**：番茄平台重视短剧改编作品

评分范围：0-100分
- 70分以上：高适配度
- 40-69分：中等适配度
- 40分以下：低适配度

### 2. 流量潜力分析

根据章节数量和内容特点，评估流量潜力：

- **内容不足**（<10章）：建议至少完成30章后再发布
- **可测试发布**（10-29章）：可小范围测试市场反应
- **适合正式发布**（≥30章）：建议正式发布并保持稳定更新
- **大量储备**（≥50章）：可采用爆更策略吸引读者

### 3. 平台特点

番茄小说网（字节跳动旗下）的特点：

- 重视短剧改编潜力
- 偏好快节奏爽文
- 系统流小说受欢迎
- 都市职场类型有市场

## 配置说明 / Configuration

平台信息存储在 `config/novel_config.json` 中的 `platforms` 字段：

```json
{
  "platforms": {
    "fanqie": {
      "name": "番茄小说网",
      "status": "待发布",
      "has_traffic": false,
      "traffic_data": {
        "last_checked": null,
        "views": 0,
        "comments": 0,
        "favorites": 0
      }
    }
  }
}
```

### 更新流量数据

如果小说已发布到番茄小说网，可以手动更新配置文件中的流量数据：

```json
{
  "platforms": {
    "fanqie": {
      "status": "已发布",
      "has_traffic": true,
      "traffic_data": {
        "last_checked": "2026-01-30",
        "views": 10000,
        "comments": 500,
        "favorites": 300
      }
    }
  }
}
```

## 常见问题 / FAQ

### Q: 如何更新流量数据？
A: 编辑 `config/novel_config.json` 文件中的 `platforms.fanqie.traffic_data` 字段。

### Q: 工具是否会自动抓取番茄小说网的数据？
A: 不会。当前版本需要手动更新流量数据。工具主要用于分析小说的适配度和潜力。

### Q: 适配分数如何计算？
A: 基于类型匹配（30分）、标签匹配（每个15分）、短剧改编特性（25分）等因素综合计算。
