# 独家视界：谎言必须死
# Exclusive Vision: Lies Must Die

> 一部融合公关行业、鉴谎系统与商业阴谋的短剧剧本类小说

## 📖 项目简介

这是一个短剧剧本类小说创作项目，讲述了公关女王林浅获得鉴谎系统后，发现丈夫陆景深的真面目，并与资本阴谋展开对抗的故事。

**核心卖点**：鉴谎系统视网膜弹窗 - 每章3-5次系统UI描写，完美适配短剧特效

## 📊 项目规划

- **总字数**：200,000 - 250,000 字
- **总章节**：100 - 120 章（完结）
- **单章字数**：1,800 - 2,000 字（严格控制）
- **结构要求**：开头有冲突 + 中间有反转 + 结尾有钩子

## 🎯 核心设定

- **主角**：林浅（公关女王 + 鉴谎系统）
- **反派**：陆景深（伪君子丈夫 + 资本操控者）
- **系统**：视网膜弹窗鉴谎系统（Truth Eye）

## ✍️ 写作原则

### ✅ 推荐
- 直接写动作
- 对话推动情节
- 系统UI弹窗（每章3-5次）
- 短句、快节奏

### ❌ 禁止
- 大段环境描写
- 抒情式心理描写
- 冗长的过渡段落

## 🎬 短剧改编定位

本作品专为短剧改编设计：
- 每个场景都可直接拍摄
- 系统弹窗可作为特效呈现
- 快节奏、强冲突
- 适合3-5分钟单集

## 📅 三卷结构

### 第一卷：觉醒（1-40章）
系统觉醒、发现真相、初步对抗

### 第二卷：博弈（41-80章）
深入调查、商业战争、关系裂痕

### 第三卷：终局（81-120章）
真相大白、最终对决、故事完结
